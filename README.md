# 项目简介

## 核心功能
- 数据爬取
- 数据排序
- 数据详情总览
- 小说数据可视化
- 搜索查询
- 分页查询

## 适用场景
- 小型或个人小说数据库管理
- 网络文学爱好者数据分析
- 教育机构辅助教学资源整理

## 技术特点
- 采用Python Flask作为后端框架
- 使用HTML、CSS、JavaScript、jQuery进行前端开发
- 数据库支持MySQL
- 支持Python 3.0以上版本
- 提供Vscode和Navicat等工具使用说明
